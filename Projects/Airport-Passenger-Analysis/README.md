# Airport Passenger Analysis (SQL & Power BI)

## Overview
This is an end-to-end data analytics project built using the same dataset.
SQL was used for data analysis and Power BI was used to create an interactive dashboard.

## Tools Used
- MySQL
- Power BI
  

## Workflow
- Raw data analysis using SQL queries
- Aggregations and insights generation
- Dashboard creation in Power BI

## Key Insights
- Monthly passenger trends
- City-wise passenger comparison
- Peak passenger periods
